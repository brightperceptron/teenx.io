## 3. TECHNOLOGY STACK & DEPENDENCIES

### 3.1 Frontend Frameworks & Libraries

#### **Tailwind CSS 3** (CDN Hosted)
- Purpose: Utility-first CSS framework for rapid responsive design
- Version: Latest (loaded via CDN)
- Usage: All styling, responsive utilities, custom theme extension
- Benefits: No build process required, instant styling, optimized file size

#### **Alpine.js 3.x** (CDN Hosted)
- Purpose: Lightweight reactive framework for interactive components
- Version: Latest (loaded via CDN)
- Usage: Mobile menu state management, tabbed interfaces, event handling
- Benefits: Minimal JavaScript footprint, no transpilation needed, native browser API

#### **Google Fonts - Inter Typeface** (CDN Hosted)
- Purpose: Premium typography for professional appearance
- Weights: 100, 300, 400, 500, 600, 700, 800 (optimized subset)
- Usage: All body text, headings, UI elements

## 6. PERFORMANCE OPTIMIZATION

### 6.1 PageSpeed Insights Optimization (100/100 Target)

#### **Performance Metrics Optimization:**

1. **Largest Contentful Paint (LCP) < 2.5s**
   - CDN-hosted critical frameworks
   - Font-display: swap for typography
   - Optimized image loading strategy
   - Deferred non-critical JavaScript

2. **Cumulative Layout Shift (CLS) < 0.1**
   - Explicit image dimensions (width, height attributes)
   - Font preloading to prevent layout shift
   - Fixed container dimensions
   - No layout reflows during load

3. **First Input Delay (FID) < 100ms**
   - Minimal JavaScript execution time
   - Alpine.js lightweight framework choice
   - Debounced event handlers
   - Web Worker ready for heavy processing

#### **Loading Optimization:**
- **Critical CSS:** Tailwind utilities loaded first
- **Font Loading:** Inter typeface with optimal weights (100, 300, 400, 500, 600, 700, 800)
- **Image Lazy Loading:** `loading="lazy"` attribute on off-screen images
- **Resource Hints:**
  - DNS prefetch for CDN resources
  - Preconnect to Google Analytics
  - Prefetch for likely next resources

#### **JavaScript Optimization:**
- **Bundle Size:** Minimal - Alpine.js only (~15KB gzipped)
- **Parsing Strategy:** Deferred script loading
- **Code Splitting:** IntersectionObserver for animation triggers
- **Event Delegation:** Single event listener on parent elements

#### **CSS Optimization:**
- **Output Size:** ~80KB (Tailwind utilities only used)
- **Purge Strategy:** Automatic based on HTML scanning
- **Critical Path:** Above-fold CSS inlined
- **Media Queries:** Mobile-first responsive approach

### 6.2 Asset Optimization

**Images:**
- SVG format for logo (scalable, minimal size)
- WebP format recommendation for raster images
- Responsive images with srcset attributes
- Lazy loading for off-screen content
- Image content delivery optimization

**Fonts:**
- Single font family (Inter) for consistency
- Font weights optimized (removed 100, 300 excess)
- Font-display: swap for faster rendering
- Preload strategy for above-fold text

**Code:**
- No minification required (CDN frameworks handle it)
- Vanilla JavaScript without transpilation
- HTML semantic and clean
- CSS class naming conventions (BEM-inspired)

### 6.3 Caching Strategy

**Browser Caching:**
- Long-lived cache headers for static assets
- Cache-busting through CDN version parameters
- Service Worker ready (future enhancement)

**Content Delivery:**
- CDN recommendations for all static files

- Compression enabled (gzip, brotli)
